## 🎬 스크롤 시퀀스 애니메이션 (Scroll-Jacking)

### 기술 개요
**미래 비전(Future Vision)** 섹션에서 Apple과 Tesla 제품 페이지에서 볼 수 있는 고급 스크롤 시퀀스 애니메이션을 구현했습니다.

### 핵심 기능
1. **True Scroll-Jacking**
   - `wheel` 이벤트를 가로채서 기본 스크롤 동작 차단 (`event.preventDefault()`)
   - 사용자가 스크롤해도 화면이 고정되어 있는 동안 이미지만 변경
   - 100% 완료 후에만 다음 섹션으로 이동

2. **듀얼 이미지 블렌딩**
   - 두 개의 이미지(와이어프레임, 실사)를 Canvas에서 블렌딩
   - 진행도에 따라 와이어프레임이 서서히 사라지고 실사가 나타남
   - 고급 시각 효과: 발광, 블러, 스케일, 플래시
   - 고해상도 디스플레이 지원

3. **시각 효과**
   - **와이어프레임 발광**: 시작 시 밝게 빛나는 네온 효과
   - **점진적 투명도**: opacity 1 → 0으로 부드럽게 사라짐
   - **블러 효과**: 70% 지점부터 약간의 블러 적용
   - **스케일 애니메이션**: 와이어프레임이 5% 확대되며 사라짐
   - **플래시 효과**: 50% 지점에서 청록색 플래시

4. **스크롤 진행도 추적**
   - 누적 스크롤 델타(deltaY)를 계산하여 애니메이션 진행도 결정
   - 0-100% 진행도 바 실시간 업데이트
   - 스크롤 잠금 인디케이터 표시 (🔒 Scroll Locked)

5. **모바일 지원**
   - `touchmove` 이벤트로 터치 스크롤 지원
   - 터치 Y좌표 델타를 계산하여 동일한 방식으로 애니메이션 제어

6. **양방향 스크롤 지원** 🔄
   - **아래로 스크롤**: 와이어프레임 → 실사 (0% → 100%)
   - **위로 스크롤**: 실사 → 와이어프레임 (100% → 0%)
   - 진행도를 자유롭게 앞뒤로 조절 가능
   - 0%에서 위로 스크롤 시 이전 섹션으로 자동 이동
   - 100%에서 아래로 스크롤 시 다음 섹션으로 자동 이동

7. **자동 섹션 전환**
   - 애니메이션 100% 완료 시 자동으로 다음 섹션으로 smooth scroll
   - 0%에서 위로 가면 이전 섹션으로 이동
   - 스크롤 잠금 자동 해제

### 파일 구조
- **`js/scroll-sequence.js`**: 스크롤 시퀀스 로직 전체
- **`css/scroll-layers.css`**: 시각 효과 및 스타일링
- **`index.html`**: Canvas 요소 및 진행도 UI

- **명확한 전환**: GM 웹사이트처럼 배경색 변화가 확실히 보임

## 🔗 YouTube API 실시간 연동

### 설정 방법 (5분 소요)

#### 1단계: Google Cloud Console에서 API 키 발급
1. [Google Cloud Console](https://console.cloud.google.com/) 접속
2. 새 프로젝트 생성 (예: "GMTCK-Analytics")
3. **YouTube Data API v3** 활성화
4. **API 키** 생성 및 복사

#### 2단계: 채널 ID 확인
1. [YouTube Studio](https://studio.youtube.com/) 접속
2. 설정 → 채널 → 고급 설정
3. **채널 ID** 복사

#### 3단계: 설정 파일 수정
```javascript
// js/youtube-config.js 파일 열기
const YOUTUBE_CONFIG = {
    API_KEY: '여기에_발급받은_API_키_입력',
    CHANNEL_ID: '여기에_채널_ID_입력',
    MAX_RESULTS: 50
};
```

#### 4단계: 사용
1. `admin-analytics.html` 접속
2. **"📺 YouTube 동기화"** 버튼 클릭
3. 실시간 데이터가 자동으로 데이터베이스에 저장됨!
